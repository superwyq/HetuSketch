# 核心功能规格

## 创作工作台

- VSCode 风布局：标题栏、活动栏、主侧边栏、编辑器工作区、辅助侧边栏、底部面板、状态栏。
- 支持多 Tab、编辑器分割、窗格 Sash 拖拽、布局持久化。
- 快捷键：`Ctrl+B`、`Ctrl+J`、`Ctrl+\\`、`Ctrl+Shift+H`。
- 章节可进入“剧情画布”，用于在正文写作前编排事件链，并支持返回章节编辑器。

## 设定与作品管理

- 设定集：创建、编辑、删除、绑定书目。
- 作品：创建、编辑、删除、导入、导出。
- 书目：创建、编辑、删除、绑定设定集。
- 分卷与章节：三级树、章节状态、字数统计、正文 Markdown。

## 设定数据库

- 角色：外貌、性格标签、能力、背景、行为红线、关系、自定义字段。
- 世界观：类别、详细描述、规则红线、关联关系、自定义字段。
- 伏笔：埋设章节、描述、预期回收章节、状态、关联角色。

## 剧情画布

- 每个章节可绑定一个剧情画布文件：`books/<bookId>/plotboards/<chapterId>.plotboard.json`。
- 支持剧情卡创建、编辑、删除、拖拽、框选、缩放、平移、撤销/重做和 Ctrl+S 保存。
- 剧情卡支持类型、客观事实、timecode、POV、地点、绑定素材、L3 状态增量、情感基调、详略等级和 AI 生成说明。
- 支持 sequence、causal、parallel、flashback、conditional 五类卡片连线。
- 素材库可绑定角色、世界观、线索、章节和模板，画布只保存引用 ID。
- 支持三幕式、推理揭示链、群像交叉线模板插入。
- 支持单卡、选区、全章、续写、重写五种生成模式；生成前保留旧正文快照，生成结果写入章节 Markdown。
- LLM 未配置时，剧情画布生成降级为本地确定性叙事编译器。
- State Diff 建议必须由用户确认、修改或拒绝后才写入章节状态快照。
- 支持时间线、角色状态、行为红线、世界规则、伏笔顺序和章节衔接校验，并可回跳剧情卡；有正文时尽量定位 Markdown 段落。
- 支持多 POV 状态轴、因果传播高亮、Markdown 大纲导出和 SVG 图片导出。

## 搜索与速查

- 标题栏全局搜索，220ms 防抖。
- SQLite FTS5 搜索角色、世界观、伏笔、作品等内容。
- 悬浮速查窗支持最近访问与搜索。
- 剧情画布保存后写入派生索引，可通过索引重建恢复。

## 逻辑校验

- 基础校验：角色红线、世界规则、未回收伏笔提醒。
- AI 增强校验：RAG 上下文 + LLM 语义判断。
- 剧情画布校验：时间线、角色状态、行为红线、世界规则、伏笔顺序、章节衔接。
- 降级策略：LLM/Embedding 未配置时保留本地结果。

## AI 与 RAG

- 支持 OpenAI 兼容与 Anthropic 风格接口。
- LLM 与 Embedding 可独立配置。
- API Key 主进程加密存储，不回传渲染端。
- RAG 支持 `fts`、`vector`、`hybrid` 三种检索模式。
- 剧情画布 AI 生成使用“赛博史官”约束：只依据剧情卡事实、素材设定、状态快照和用户指令生成正文，不自动改变设定事实。
