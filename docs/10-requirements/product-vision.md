# 产品愿景

HetuSketch（河图速写助手）是面向小说创作者的本地优先型设定管理、剧情编排与逻辑校验桌面应用。

## 产品定位

HetuSketch 是“逻辑监工”和“结构化创作工作台”，不是“代笔枪手”。核心目标是帮助作者记录、管理、检索、编排和校验角色、世界观、情节线索与伏笔，降低长篇创作中的记忆负担和逻辑冲突风险。

剧情画布补足设定库与章节正文之间的中间层：作者设计事实、顺序、因果和状态变化，AI 只作为“赛博史官”把已确定的事件链写成正文草稿。

## 目标用户

| 用户 | 诉求 |
| --- | --- |
| 长篇小说作者 | 管理复杂角色、规则和伏笔，维护前后一致性。 |
| 同人文作者 | 对照原作设定，避免 OOC 和世界观偏离。 |
| 剧本 / 群像创作者 | 梳理人物关系、剧情线和冲突点。 |
| 世界观设定爱好者 | 结构化管理地理、势力、能力体系等设定。 |
| 悬疑推理作者 | 追踪线索埋设、强化、误导和回收顺序。 |
| 碎片化创作者 | 先记录关键事件，再通过画布补齐前因后果。 |

## 核心价值

- 本地优先：作品、设定、画布、章节和快照默认保存在用户本机。
- 离线可用：设定管理、搜索、剧情画布编辑、导出和基础校验不依赖 AI。
- AI 可选增强：用户配置 LLM / Embedding 后启用语义校验、补全、RAG 和剧情画布正文生成。
- 一体化工作台：VSCode 风布局整合设定管理、正文创作、剧情画布、校验和速查。
- 面向长篇创作：支持设定集、作品、书目、分卷、章节、角色、世界观、伏笔、剧情卡、状态快照的结构化链路。
- 用户掌控：AI 生成正文和 State Diff 均为建议，关键事实源变更必须由用户确认。
