# 产品愿景

HetuSketch（河图速写助手）是面向小说创作者的本地优先型设定管理与逻辑校验桌面应用。

## 产品定位

HetuSketch 是“逻辑监工”，不是“代笔枪手”。核心目标是帮助作者记录、管理、检索和校验角色、世界观、情节线索与伏笔，降低长篇创作中的记忆负担和逻辑冲突风险。

## 目标用户

| 用户 | 诉求 |
| --- | --- |
| 长篇小说作者 | 管理复杂角色、规则和伏笔，维护前后一致性。 |
| 同人文作者 | 对照原作设定，避免 OOC 和世界观偏离。 |
| 剧本 / 群像创作者 | 梳理人物关系、剧情线和冲突点。 |
| 世界观设定爱好者 | 结构化管理地理、势力、能力体系等设定。 |

## 核心价值

- 本地优先：作品和设定默认保存在用户本机。
- 离线可用：设定管理、搜索和基础校验不依赖 AI。
- AI 可选增强：用户配置 LLM / Embedding 后启用语义校验、补全和 RAG。
- 一体化工作台：VSCode 风布局整合设定管理、正文创作、校验和速查。
- 面向长篇创作：支持设定集、作品、书目、分卷、章节、角色、世界观、伏笔的结构化链路。
