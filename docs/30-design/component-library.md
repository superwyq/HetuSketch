# 组件库规范

当前 UI 基于 React + Ant Design，并在全局样式中形成 HetuSketch 工作台组件语义。

## 工作台组件

| 组件 | 职责 |
| --- | --- |
| TitleBar | 品牌、命令中心、作品选择、速查、置顶、设置、主题、窗口控制。 |
| ActivityBar | 左侧活动入口，支持拖拽排序与实体色激活指示。 |
| PrimarySidebar | 当前活动对应的树或列表。 |
| EditorWorkbench | 多 Tab、多分割编辑区域。 |
| SecondarySidebar | AI Chat / Outline / Timeline 辅助视图。 |
| BottomPanel | AI 提示、角色、世界观、线索、输出等底部视图。 |
| StatusBar | 当前作品、光标、字数、面板状态、编码等状态信息。 |
| Sash | 可拖拽分隔条。 |

## 页面组件

| 页面 | 职责 |
| --- | --- |
| DashboardPage | 总览、统计、最近访问和能力入口。 |
| ProjectsPage | 作品创建、编辑、删除、导入导出。 |
| EntriesPage | 角色、世界观、伏笔复用管理页。 |
| WritingStudioPage | 章节树、Markdown 编辑、预览、校验和剧情画布入口。 |
| PlotboardPage | 剧情画布；承载素材库、剧情卡画布、连线、属性面板、生成面板、校验面板和状态栏。 |
| SettingsPage | 主题、字体、AI、提示词、技能、HTTP 工具配置。 |
| QuickLookupPage | 悬浮速查窗内容。 |

## 剧情画布组件语义

| 组件/区域 | 职责 |
| --- | --- |
| 顶部工具栏 | 保存、撤销/重做、生成模式、生成说明、生成正文、取消、逻辑校验、导出、返回章节。 |
| 素材库 | 角色、世界观、线索、章节和模板素材搜索、点击绑定、拖拽绑定。 |
| 中央画布 | React DOM 剧情卡、SVG 连线、空状态引导、缩放平移、框选。 |
| 剧情卡 | 显示类型、timecode、标题、事实摘要、绑定素材、校验状态和连线锚点。 |
| 属性面板 | 编辑卡片基础信息、素材绑定、线索使用方式、生成设置、L3 状态增量；或编辑连线关系。 |
| 多 POV 状态轴 | 按 POV 汇总卡片和状态字段，支持跳转卡片。 |
| 生成面板 | 展示生成进度、Markdown 输出、State Diff 确认/修改/拒绝。 |
| 校验面板 | 展示分类 finding、正文段落定位、线索 resolved 更新入口。 |
| 底部状态栏 | 展示卡片数、连线数、冲突数、未回收线索、选中数、缩放和快捷键提示。 |

## 组件规则

- 设定条目列表中的类型图标使用实体识别色。
- 校验结果使用风险等级色。
- 危险操作使用确认弹窗，不让行级删除按钮常驻红色。
- 图标按钮必须有明确语义，主要操作与辅助操作区分视觉权重。
- 剧情画布中的卡片事实应突出“发生了什么”，生成说明和文学表达不应喧宾夺主。
- State Diff、AI 生成正文和线索 resolved 更新必须表现为“建议/待确认”，避免自动覆盖事实源。
