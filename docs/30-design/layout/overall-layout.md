# 工作台整体布局

工作台实现以 `src/renderer/src/App.tsx` 与 `src/renderer/src/styles.css` 为准，详细旧文档见 `../../ui-layout.md`。

## 区域

```text
┌─────────────────────────────────────────────┐
│ TitleBar                                    │
├──┬──────────────┬──┬──────────────┬──┬──────┤
│A │ Primary      │S │ Editor       │S │ Sec. │
│c │ Sidebar      │a │ Workbench    │a │ Side │
│t │              │s │              │s │      │
├──┴──────────────┴──┴──────────────┴──┴──────┤
│ BottomPanel                                 │
├─────────────────────────────────────────────┤
│ StatusBar                                   │

```

## 默认尺寸

| 区域 | 默认 | 约束 |
| --- | --- | --- |
| TitleBar | 32px | Windows；macOS 可为 22px |
| ActivityBar | 48px | 固定 |
| PrimarySidebar | 260px | 170px–500px，可折叠 |
| SecondarySidebar | 320px | 220px–500px，默认隐藏 |
| BottomPanel | 200px | 最小 80px，可折叠 |
| StatusBar | 22px | 固定 |

## 剧情画布页面布局

剧情画布作为主编辑器页面承载，不使用 Node 权限，内部采用全屏式五区布局：

```text
┌──────────────────────────────────────────────────────────────┐
│ Plotboard Toolbar                                            │
├──────────────┬───────────────────────────────┬───────────────┤
│ Material     │ POV Axis + Canvas             │ Properties    │
│ Library      │ DOM Cards + SVG Links         │ Panel         │
├──────────────┴───────────────────────────────┴───────────────┤
│ Generation Panel / Validation Panel                          │
├──────────────────────────────────────────────────────────────┤
│ Plotboard Statusbar                                           │
└──────────────────────────────────────────────────────────────┘
```

- 窄屏下优先保留中央画布可操作面积，素材库和属性面板保持可滚动。
- 画布空状态显示引导文案，提示双击创建卡片或从左侧拖入素材。
- 底部状态栏固定展示卡片数、冲突数和快捷键提示，避免用户迷失当前编辑状态。

## 持久化

工作台布局状态保存到 `localStorage`，包括尺寸、显隐、编辑器分割模式、活动栏顺序、Tab 列表和侧边栏文件夹。

剧情画布内部 `viewport` 保存到画布 JSON，选中态、撤销/重做、生成进度和校验结果只保留在页面会话。
